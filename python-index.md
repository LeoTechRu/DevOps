# Python

Корень для Python-материалов внутри `DevOps`.

## Что здесь лежит

- `courses/otus-python-developer-basic/` — канонический архив курса OTUS `Python Developer. Basic`:
  - `homework-repo/` — архив вашего репозитория с домашними заданиями;
  - `notes/course-outline.md` — программа курса по месяцам;
  - `notes/devops-track.md` — выжимка тем, которые полезны для DevOps/SRE-линии;
  - `notes/materials-index.md` — индекс внешних материалов, презентаций и кодовых примеров.
- `courses/python-testing-course/` — отдельный блок по pytest, Selenium и тестированию:
  - `materials/` — исходные материалы курса;
  - `practice/` — локальные упражнения, проверки домашних заданий и вспомогательные примеры.
- `labs/phonebook/` — канонический phonebook-lab из legacy `studyPython/13`.
- `labs/fastapi-compose-notes/` — отдельный lab по FastAPI, `Dockerfile`, `docker-compose` и `nginx`.
- `archive/study-python-misc/` — хвосты старой песочницы `studyPython`, которые не стали отдельными lab'ами.

## Что сделано с legacy `studyPython`

- консольный phonebook вынесен в `labs/phonebook/`;
- FastAPI+Compose кейс вынесен в `labs/fastapi-compose-notes/`;
- блок тестирования собран в `courses/python-testing-course/`;
- дубликаты, ранние версии и разнородные учебные куски сложены в `archive/study-python-misc/`.
