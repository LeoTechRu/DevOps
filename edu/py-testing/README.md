# Python Testing Course

Собранный блок по тестированию из legacy `studyPython`.

Структура:

- `materials/` — исходные материалы курса `PythonTest_18082025`;
- `practice/` — локальные упражнения, примеры pytest/selenium и проверки домашних заданий.

Этот каталог отделён от OTUS-архива, потому что это другой учебный трек с отдельным фокусом: тесты, практики проверки и автоматизация QA-подходов внутри Python-стека.
