# OTUS Python Developer. Basic

Канонический архив курса OTUS `Python Developer. Basic` внутри `DevOps`.

## Метаданные курса

- источник: `https://otus.ru/learning/332936/`
- публичный upstream с кодом занятий: `https://github.com/OtusTeam/BasePython/tree/PythonBasic.2025-01`
- период обучения: `30 января 2025` — `10 сентября 2025`
- статус группы: завершила обучение
- сдано ДЗ: `13 из 14`
- дата сертификата: `10 сентября 2025`

## Структура

- `homework-repo/` — ваш архив репозитория с домашними заданиями и тестами курса.
- `notes/course-outline.md` — скелет программы по месяцам.
- `notes/devops-track.md` — темы курса, которые дают ценность для текущего DevOps-вектора.
- `notes/materials-index.md` — внешний индекс методичек, кодовых примеров, презентаций и документации.

## Зачем это так разложено

Раньше курс жил в каталоге `BasePython`, что больше отражало название учебного репозитория, чем сам курс.
Теперь структура отражает:

1. это именно курс `Python Developer. Basic`;
2. `homework-repo/` — лишь один из его артефактов;
3. рядом лежит нормальный конспект, а не только решения ДЗ.

## Ограничения архива

- видео вебинаров сознательно не копируются в git и на машину;
- в `notes/` фиксируются только текстовые материалы, публичные ссылки, кодовые ориентиры и практические выводы;
- часть материалов доступна только через OTUS-кабинет, поэтому `notes/materials-index.md` хранит прежде всего стабильные внешние ссылки и ориентиры по урокам.
