Intelligent Data ERP

Архивный code snapshot ERP-контура вокруг Odoo/ERPNext, SSO и deploy-обвязки. Контур decommissioned 22 марта 2026 года и больше не поддерживается как pet-проект.

Snapshot

Полный code snapshot лежит в `snapshot/`.

Перенесён из `/int/erp` без `.git`, `openspec`, `GEMINI.md` и `.env.example`, чтобы сохранить инженерный контекст без live-runtime ownership.

Стек

Odoo, ERPNext, SSO, reverse proxy, deploy scripts.

В архиве остались compose-слой, generated конфиги Apache/Nginx, Python SSO helpers и repo-local документация.

Статус

Runtime удалён с машины.

Хостовые unit'ы, vhost'ы, docker-хвосты и связанный runtime больше не считаются живым контуром. Новая разработка возможна только под отдельный заказ.
