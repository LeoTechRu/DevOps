# ITSM Module Core

Выделенный модуль ITSM для экосистемы IntData: Flowable-процессы, перечисления,
DTO, репозитории, API и документация. В модуле размещаем весь код, который ранее
жил в `shared/itsm` и `nexus/api/services/itsm_*`. Специфический функционал
других модулей (Nexus, CRM и т.д.) подключается через публичные фасады `itsm`.
