Snapshot

Полный code snapshot лежит в `snapshot/`.

Перенесён из `/int/itsm` без `.git`, `openspec` и `GEMINI.md`, чтобы сохранить рабочий код и docs без live-runtime ownership.

Стек

Flowable, BPMN, Python service layer, migrations.

В архиве остались DTO, models, repository, config layer, Flowable adapters, db migrations и devops-скрипты.

Статус

Runtime удалён с машины.

Docker-hвосты Flowable и связанные host-level следы больше не считаются живым контуром. Новая разработка возможна только под отдельный заказ.
